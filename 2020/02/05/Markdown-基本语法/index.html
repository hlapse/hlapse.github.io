1. Markdown是什么

Markdown是一种轻量级的标记语言，它以易读易写的纯文本形式编写
Markdown语言在2004由约翰·格鲁伯（英语：John Gruber）创建。
Markdown编写的文档可以导出HTML、Word、图像、PDF、Epub等多种格式的文档。
Markdown编写的文档后缀为 .md或 .markdown。
